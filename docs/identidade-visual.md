# identidade visual - Cláudio Pequenos reparos (v2)

## Base Visual
A identidade visual do site foi ajustada 
com base no cartão do cliente,
utilizando cores que trnsmitem ação,
confiança e clareza.

## Cores Principais
- Vermelho: ação, reparo, urgéncia
- Branco: claresa e leitura
- Tons escuros: proficionalismo

## Objetivo
Aumentar impacto visual e reforça
reconhecimento da marca,
mantendo cinolicidade e rapidez no
carregamento.

## Observação
Essa identidade é aplicada sobre um
template reutilizavel,
permitindo replicação para outros clientes
com baixo esforço.
